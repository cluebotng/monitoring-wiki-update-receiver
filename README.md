# Prometheus AlertManager Wiki Receiver

This is a shim API to update Mediawiki page contents based on alert status.

Example pages:

* https://en.wikipedia.org/wiki/User:ClueBot%20NG/running
* https://en.wikipedia.org/wiki/User:ClueBot%20III/running

## Production config

* `WIKI_UPDATER_USERNAME` - Wikipedia username to use for editing
* `WIKI_UPDATER_PASSWORD` - Wikipedia password to use for editing
